## Measurement Method

This report uses a reproducible source-code proxy, not a final architecture-completion proof.

- Root: `backend/src/main/java/com/colonel/saas`
- Files: `*.java` under production source only
- Source LOC: non-empty lines excluding lines whose trimmed text starts with `//`, `/*`, or `*`
- `ddd_domain`: paths under `domain/`
- `legacy_business_entry`: paths under `service/`, `controller/`, `auth/`, `job/`, `listener/`
- `support_infra_or_dto`: mapper/entity/config/common/security/exception/thirdparty/gateway/dto/vo/testsupport
- `other`: files not matched by the buckets above

PowerShell reproduction outline:

```powershell
$root = Resolve-Path 'backend/src/main/java/com/colonel/saas'
Get-ChildItem $root -Recurse -Filter *.java -File |
  ForEach-Object { count nonblank non-comment source LOC and bucket by relative path }
```
