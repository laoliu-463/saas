## 5. 活动商品同步链路

| 维度 | 事实 |
|------|------|
| 入口 | ProductService.refreshActivitySnapshots |
| 触发 | ProductActivitySyncJob (5分钟cron) / 手动API |
| 分页 | cursor 分页, max 100 pages, batch 1-20 |
| 入库决策 | 推广中→自动入库; 非推广中→隐藏 |
| 后置 | repairLibraryStateForActivity → applyForActivityId |
| 事件 | ActivitySyncCompletedEvent |

| 风险 | 大事务; 后置修复可能超锁TTL |

## 6. 商品库展示规则

- 三阶段持久化 (P-FIX-002): 降级→处理→升级
- 唯一索引: uk_pos_one_displaying_per_product
- 优先级: forceDisplay > supportsAds > commissionRatio↓ > serviceFeeRate↑ > shelfTime↑
- 保护期: 默认3个月
- 隐藏原因: 8种常量
- **适合提取 ProductDisplayPolicy**

## 7. 商品库查询/筛选/分页

- 全量内存分页: collectSelectedLibraryProducts → sort → subList
- 筛选项: keyword/status/category/activityId/assigneeId/colonelPartnerId/supportsAds/sortBy
- **问题**: Query/Command 混合; 全量加载性能风险
