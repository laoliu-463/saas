## 风险和后续建议

1. **商品图片**: 后端 `ColonelsettlementOrder` 实体目前无 `productImage` 字段，当前显示占位。后续可从抖店 API 补充商品图片 URL。
2. **商品数量/佣金率/服务费率**: 后端实体无对应字段，当前显示 `-`。如需展示，需后端做最小 DTO 补齐（从订单已有字段读取）。
3. **字段命名统一**: 前端已做兼容（normalize），但建议后端统一字段命名规范。
4. **浮点精度**: 已修复 `0.07 * 100 = 7.000000000000001` 精度问题，使用 `Math.round(num * 10000) / 100`。
