## 零、V1 目标与三条业务链（验收主线）

**V1 目标**：跑通基础业务，**不做**独家达人、独家商家及一切相关计算、配置、事件。

| 链路 | V1 路径 |
|------|---------|
| **渠道** | 认领 → 选品 → 转链 → 寄样 → 订单 → **业绩**（默认归因 + 双轨提成） |
| **招商** | 同步活动 → 上架 → 审寄样 → 订单 → **业绩**（默认招商归因） |
| **管理** | 配人 → 配规则 → 各域读配置 |

**业绩域 V1 约定**：`最终渠道 = 默认渠道`，`最终招商 = 默认招商`（含个别品负责人）；不实现独家覆盖、不跑独家月度任务、不订阅独家资格事件。

**领域设计文档**暂不必大改；实现与测试以本表为准，V2 再启用达人域/业绩域中的独家章节。
