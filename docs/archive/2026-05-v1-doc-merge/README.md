## 当前技术实现

| 层级 | 当前实现 |
|---|---|
| 后端 | Spring Boot 模块化单体 |
| 数据库 | PostgreSQL |
| 缓存 | Redis |
| 前端 | Vue 3 + Vite + Pinia + Naive UI |
| 部署 | Docker Compose |
| 三方隔离 | Gateway 契约，按 Test / Real 切换实现 |

补充说明：

- 新版需求包中的 `FastAPI + Celery` 属于原始技术落地建议，不是当前仓库的必须迁移目标。
- 当前文档不再使用“必须迁移到 FastAPI / Celery”这类表述。

## 当前环境简表

| 环境 | 用途 | 当前口径 |
|---|---|---|
| `local-mock` | 历史 Mock 口径与回滚参考 | 不再作为独立主线环境扩展 |
| `test` | 系统功能 / 权限测试 / 本地 Mock 联调 | 当前默认功能验证环境 |
| `real-pre` | 真实 SDK / 真实上游联调 | 当前三方联调环境 |
| `prod` | 正式环境规划 | 本期不展开 |
