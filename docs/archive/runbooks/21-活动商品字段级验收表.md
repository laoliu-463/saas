# 活动商品字段级验收表（PRODUCT-05）

**版本**：V1 收口（2026-05-23）  
**对照文档**：`docs/requirements/domain-design/02-商品域.md` §9.4~9.5、§4 展示规则、§10.2 合作方/活动商品接口  
**代码主路径**：`GET /api/colonel/activities/{activityId}/products`、`GET /api/colonel/activities/{activityId}/products/{productId}`

---

## 1. 验收口径说明

| 类别 | V1 通过标准 | 未纳入 V1 |
| ---- | ----------- | --------- |
| 活动商品排序 | 本地快照列表支持 `sortBy=default\|latest`；`default` 为置顶 > 转链 > 佣金 > 同步时间 | 上游 `sortType` 与本地排序叠加策略未统一文档化 |
| 审核资料 | `audit_payload` 持久化；详情返回完整 `auditSupplement` + `promotionMaterialPack`；列表返回 `auditSupplementSummary` + `auditSupplementComplete`；货品/商品标签 `goodsTags/productTags` 可保存、摘要返回并参与多选筛选 | - |
| 展示规则 | 列表/详情返回 `displayMark`/`displayMarkLabel`；商品库侧按 `selected_to_library` 与置顶规则排序（见 P-09） | 3 个月保护期全量重算、优势覆盖自动切换 |

## 4. 商品库展示规则（P-09 简化对照）

| 文档规则 | V1 实现 | 验收方式 |
| -------- | ------- | -------- |

| 投流 > 高佣 > 晚上架 | 同上 | 同上 |

## 6. 结论（PRODUCT-05）

- **可勾选通过**：活动商品列表/详情的核心业务字段、审核资料读写、`goodsTags/productTags` 多选筛选、简化展示标记、本地列表排序与商品库展示简化规则均有代码与测试锚点。
- **仍记缺口**：展示规则全量引擎（保护期/优势覆盖自动重算）归入 V2，不在本表标为 V1 通过。
