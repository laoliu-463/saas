## 三、发现并修复的问题

### Bug #1: SampleController SQL 语法错误 (已修复)

**严重级别**: HIGH — 运行时异常

**症状**: `GET /api/samples/board` 返回 500

**根因**:
- `SampleController.getSampleBoard()` 创建 `QueryWrapper` 并调用 `wrapper.orderByDesc("create_time")`
- `SampleRequestMapper.findPageWithScope()` 的 SQL 模板将 `ew.sqlSegment` 插入 WHERE 子句
- 当 DataScope 不添加条件时 (ALL scope)，`sqlSegment` 包含 `ORDER BY create_time DESC`
- 生成的 SQL: `WHERE sr.deleted = 0 AND ORDER BY create_time DESC ORDER BY sr.create_time DESC` — 语法错误

**修复**: 删除 `SampleController.java:261` 的 `wrapper.orderByDesc("create_time")`，因为 Mapper SQL 模板已包含 `ORDER BY sr.create_time DESC`

**验证**: 修复后 `/api/samples/board` 返回 200，7 个看板列数据正常

### Bug #2: /v3/api-docs 返回 401 (已确认非问题)

**症状**: 之前测试时 `/v3/api-docs` 返回 401

**根因**: 测试时未加 `/api` 前缀。Spring Security 已正确配置 `/v3/api-docs/**` 到 permitAll。实际 URL 为 `/api/v3/api-docs`，返回 200。

**状态**: 无需修复
