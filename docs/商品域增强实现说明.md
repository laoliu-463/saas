# 商品域增强实现说明

## 范围

本次增强覆盖五项能力：展示规则引擎 V3、团长主数据、商品库全量筛选、领域事件 Outbox、抖店 quick_sample_apply 适配层。

## 技术栈

Spring Boot + PostgreSQL + Redis；前端 Vue 3 + Vite + Naive UI。未引入 Kafka/RabbitMQ。

## 关键模块

| 能力 | 主要类 |
|------|--------|
| 展示规则 | `ProductDisplayRuleService`, `AdminProductDisplayService`, `ProductDisplayRuleJob` |
| 审计 | `ProductDisplayAuditService`, `product_display_audit_log` |
| 团长主数据 | `ColonelPartnerSyncService`, `ColonelPartnerAdminService` |
| Outbox | `OutboxEventAppender`, `ProductDomainEventPublisher`, `DomainEventDispatcherJob` |
| 快速寄样 | `DouyinQuickSampleGateway`, `ProductQuickSampleService` |
