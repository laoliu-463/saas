## 一、V1 范围

### V1 做

- Y-01 默认归因业绩（最终 = 默认；无独家覆盖）
- Y-02 双轨提成（预估轨 + 结算轨）
- Y-03 提成基数 = 服务费收益
- Y-06 退款 / 失效冲正
- Y-07 业绩列表 + 指标卡片
- Y-08 订单列表批量补全业绩字段
- Y-10 全局提成比例读取配置域

### V1 简化

- Y-09 管理员重算单笔订单

### V1 不做

- Y-04 毛利
- Y-05 商品负责人变更后未结算单重算

## 三、与领域设计的主要偏差

| 类型 | 条目 | 说明 |
|---|---|---|
| 范围差异 | 独家覆盖逻辑 | 领域设计里有独家覆盖描述，但当前 V1 明确只按默认归因 |
| 少做 | Y-09 单笔重算入口 | 当前没有明确单独入口，先按简化处理 |
| 明确后移 | Y-05 | 负责人变更重算不纳入当前 V1 主范围；Y-04 毛利已按 2026-06-05 用户决策纳入经营指标 |

## 四、状态机与规则检查

| 检查项 | 当前情况 | 结论 |
|---|---|---|
| 默认归因 | 当前按默认渠道 / 默认招商收口 | ✅ |
| 双轨提成 | 已按预估 / 结算两轨计算 | ✅ |
| 退款 / 失效冲正 | 已跟随订单状态变化 | ✅ |
