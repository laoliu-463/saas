# Harness 维护资料

这里保存从 `harness/` 移出的历史规则、任务记录和工程配置，仅供 CI/CD 维护者、值班人员和迁移审查使用，不是普通开发入口。

- `legacy-rules/`：旧版规则、领域指令、技能和状态快照；当前执行以 `harness/policy`、`harness/runbooks`、`harness/checks` 和 `harness/scripts` 为准。
- `engineering/`：Issue tracker、标签和上下文维护配置。
- `tasks-history/`：历史任务卡和交接记录；新任务使用 GitHub Issues。
- `templates-history/`：旧模板；当前仅保留 `harness/templates/` 三种模板。
