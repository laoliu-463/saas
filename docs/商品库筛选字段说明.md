# 商品库筛选字段说明

## 筛选项（AND 组间 / 类目与标签 OR 组内）

| 参数 | 说明 | 数据来源 |
|------|------|----------|
| keyword | 名称/ID/店铺 | 列 + 快照 |
| categories | 类目多选 OR | snapshot.category |
| shopKeyword | 商家名称 | snapshot.shop_name |
| colonelName | 团长名称 | colonel_partner 映射 |
| published | 已发布 | operation_state |
| cooperationType | 合作类型 | supplement_info |
| hasSample / freeSample | 寄样 | snapshot / supplement |
| assigneeId | 招商负责人 | operation_state |
| recruitActivityId/Name | 招商活动 | snapshot |
| livePriceMin/Max | 直播价格 | supplement_info.livePrice |
| commissionMin/Max | 佣金率 | snapshot |
| sampleSalesMin/Max | 寄样销售额 | supplement_info |
| supportsAds | 投流 | audit tags |
| notInLibrary | 仅未加入货盘 | supplement_info |
| dedup | 选品去重 | 列表层 scope |
| materialDownload | 素材下载 | audit checkbox |
| exclusivePrice / productChain / handCard / doubleCommission | 标签 | product tags |
| goodsTags / productTags | 标签多选 OR | audit |

## supplement_info 建议字段

见任务规格 JSON（cooperationType、sampleType、livePrice、supportsAds 等）。

## 后端入口

`ProductService.SelectedLibraryFilter` + `matchesSelectedLibraryFilters`；`GET /products` 查询参数。
