# 06-部署与对接计划

更新时间：2026-04-21

## 一、本地部署指南 (Local)

### 1. 前置要求
- Docker Desktop
- Node.js 18+
- Maven 3.8+

### 2. 快速启动
```bash
# 1. 启动数据库
docker-compose up -d db

# 2. 运行后端 (Test Profile)
cd backend
mvn spring-boot:run -Dspring-boot.run.profiles=test

# 3. 运行前端
cd frontend
npm install
npm run dev
```

## 二、生产部署方案 (Real)

### 1. 环境拓扑
- **应用服务器**：2核4G (推荐)
- **数据库**：RDS MySQL 8.0
- **缓存**：Redis (可选，用于 Token 缓存)

### 2. 切换至 Real 环境
1. 修改 `backend/src/main/resources/application.yml`：
   ```yaml
   spring:
     profiles:
       active: real
   ```
2. 配置 `application-real.yml` 中的 AppKey 与 AppSecret。
3. 确保外网 Webhook 能够访问 `/api/callback/douyin`。

## 三、第三方对接计划

### 1. 抖店开放平台
- **第一步**：申请“服务商”角色，创建应用并获取授权。
- **第二步**：联调 `DouyinAuthGateway` 获取 ShopID 权限。
- **第三步**：完成 `Activity` 与 `Product` 接口的真实数据透传。

### 2. 物流平台
- **当前方案**：本项目内置简单的物流轨迹模拟。
- **扩展计划**：对接快递100或菜鸟 SDK，实现真实单号自动签收更新。

## 四、升级与迁移

1. **DB 迁移**：使用增量 SQL 脚本 (`alter-*.sql`) 进行平滑升级。
2. **数据清理**：切入生产环境前，需执行 `POST /api/test/reset` 清理演示数据。
