# real-pre 联调手册

> 范围标记：`V1 必做`、`V1 简化`、`V1 不做`、`V2 预留`、`历史归档`。

## 联调目标

- [V1 必做] real-pre 是上线前真实联调环境。
- [V1 必做] real-pre 验收不得使用 mock 数据冒充真实闭环。

## 环境前置

| 项 | 要求 | 验证方式 | 范围 |
| --- | --- | --- | --- |
| test 开关 | `APP_TEST_ENABLED=false` | 环境变量、启动日志 | V1 必做 |
| 抖音 mock 开关 | `DOUYIN_TEST_ENABLED=false` | 环境变量、启动日志 | V1 必做 |
| 端口 | 前端 `3001`、后端 `8081`、PostgreSQL `5433`、Redis `6380` | Docker ps、启动日志 | V1 必做 |
| Token | 真实授权和有效 Token | preflight、后端日志 | V1 必做 |
| 数据 | 真实商品、订单、达人或明确 BLOCKED 证据 | API/SQL/第三方响应 | V1 必做 |

## 命令

| 命令 | 用途 | 状态 |
| --- | --- | --- |
| `npm run e2e:real-pre:p0:preflight` | 环境、Token、基础接口预检 | V1 必做 |
| `npm run e2e:real-pre:p0` | real-pre P0 浏览器验收 | V1 必做 |

## 验收项

| 项 | 真实证据 | 不通过条件 | 范围 |
| --- | --- | --- | --- |
| Token | 获取 / 刷新日志、第三方响应 | mock Token 或过期 Token | V1 必做 |
| 商品同步 | 第三方商品响应、商品入库 | 手工 mock 商品冒充同步 | V1 必做 |
| 转链 | 第三方转链响应、`pick_source_mapping` | 无映射记录 | V1 必做 |
| 订单同步 | 第三方订单响应、订单入库 | mock 订单冒充真实回流 | V1 必做 |
| 寄样交作业 | 订单已同步事件和寄样状态 | 人工改状态无证据 | V1 必做 |
| 业绩计算 | 业绩明细、汇总、输入来源 | 订单域直接算提成 | V1 必做 |

## 状态记录

- [V1 必做] PASS：真实证据完整。
- [V1 必做] BLOCKED：外部权限、Token、限流、样本缺失阻塞，需记录请求、响应、时间和下一步。
- [V1 必做] FAIL：系统内可复现失败，需记录最小复现和日志。
- [V1 简化] PASS_NEEDS_CLEANUP：主链路通过但日志、数据或文档需清理。
