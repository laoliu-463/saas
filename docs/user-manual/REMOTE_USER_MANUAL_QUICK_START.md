# 抖音团长 SaaS 系统 — 快速上手指南

> 一页纸版本，适合发群或快速了解系统。完整手册见《REMOTE_USER_MANUAL.md》。

---

## 系统做什么

帮助团长团队管理商品、达人、寄样、订单和业绩，替代手工 Excel 和微信群管理。

## 如何登录

1. 打开浏览器（推荐 Chrome / Edge），访问系统网址（管理员提供）。
2. 域名访问：`https://real-pre.<域名>` → 自动跳转到登录页。
3. 直接访问登录页：`https://real-pre.<域名>/login`。
4. 输入用户名和密码，点击“登 录”。
5. 登录成功后自动跳转到对应角色的首页。

> 忘记密码或账号被锁，请联系管理员重置。

## 三条主线

| 业务线 | 流程 | 主要角色 |
|--------|------|----------|
| 渠道线 | 认领达人 → 选品 → 复制讲解 → 寄样 → 看业绩 | 渠道、渠道组长 |
| 招商线 | 同步活动 → 上架商品 → 审核寄样 → 看业绩 | 招商、招商组长 |
| 管理线 | 建账号 → 分角色 → 配规则 → 看数据 | 管理员 |

## 预置账号（仅供测试/参考）

| 用户名 | 角色 | 数据范围 |
|--------|------|----------|
| `admin` | 管理员 | 全部数据 |
| `biz_leader` | 招商组长 | 本组 |
| `biz_staff` | 招商专员 | 本人 |
| `channel_leader` | 渠道组长 | 本组 |
| `channel_staff` | 渠道专员 | 本人 |
| `ops_staff` | 运营 | 本人 |

> 正式使用请联系管理员创建个人账号，不要共用测试账号。

## 角色速查

| 角色 | 数据范围 | 日常操作 |
|------|----------|----------|
| 管理员 | 全部 | 建账号、配角色、配规则 |
| 招商组长 | 本组 | 同步活动、分配商品、审核寄样 |
| 招商专员 | 本人 | 补充商品、审核寄样 |
| 渠道组长 | 本组 | 认领达人、选品、申请寄样 |
| 渠道专员 | 本人 | 认领达人、选品、申请寄样 |
| 运营 | 本人 | 录入物流、处理发货 |

## 每日必做

1. **渠道**：进商品库 → 复制讲解 → 发给达人 → 申请寄样。
2. **招商**：同步活动 → 补充商品 → 审核寄样。
3. **运营**：处理待发货寄样 → 录入物流。
4. **管理层**：看数据看板 → 关注订单额和服务费收益。

## 重要提醒

- **必须使用"复制讲解"生成的推广链接**，否则订单无法归因到渠道。
- 订单有 10 分钟到 1 小时同步延迟，不是付款后立刻出现。
- 修改权限后需要刷新页面或重新登录。
- 不要多人共用一个账号。

## 当前已知限制

- 结算数据暂未到位，结算轨指标显示为 0。
- 渠道归因需要真实通过系统链接产生的订单验证。
- 物流以手动录入为主。

## 遇到问题

- 登录不了 → 检查用户名密码是否正确，或联系管理员确认账号状态。
- 看不到菜单 → 联系管理员在“角色管理”中配置角色菜单权限。
- 看不到数据 → 确认角色数据范围（本人/本组/全部）。
- 订单找不到 → 等同步延迟（10 分钟 ~ 1 小时）或联系管理员。
- 寄样没自动完成 → 确认达人是否通过系统链接出单。
- 页面显示异常 → 按 Ctrl + Shift + R 强制刷新，或换 Chrome / Edge 浏览器。
