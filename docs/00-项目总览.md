# 00-项目总览

更新时间：2026-04-27

## 一、项目定位

本项目是一套面向抖音团长业务的 SaaS 系统，目标是把以下几条业务线串成统一闭环：

- 商品库与招商协同
- 推广转链与归因映射
- 订单回流与业绩归因
- 寄样履约与自动完成
- 达人 CRM 沉淀

当前项目的重点不是“功能堆积”，而是：

> 在 `test` 环境下，把核心业务跑通、跑稳、跑得可演示、可回归。

## 二、当前阶段结论

当前项目阶段正式定义为：

> `test MVP 核心闭环完成阶段`

这意味着当前系统已经具备：

- 可演示
- 可回归
- 可对外汇报
- 可作为真实联调准备基线

## 五、当前主链路

当前已经封口的主链路有五条：

### 1. 商品库主链路

商品展示 -> 商品详情 -> 复制推广链接 -> `pick_source_mapping`

### 2. 订单归因主链路

Test 订单 -> 已归因 / 未归因 -> 订单工作台 -> 看板 -> 详情排查

### 3. 寄样履约主链路

申请寄样 -> 审核 -> 发货 -> 签收 -> 待交作业 -> 出单自动完成

### 4. 达人 CRM 主链路

达人列表 -> 新增 -> 公海 / 私海 -> 认领 / 释放 -> 详情 -> 关联寄样 / 订单

### 5. Test 调试主链路

`reset / seed / 生成订单 / 发货 / 签收 / 回归验证`
