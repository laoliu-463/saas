# 00 项目总览

更新时间：2026-04-25

## 项目名称

抖店 SDK / 招商团长 SaaS 系统

## 项目目标

围绕招商团长业务，建设一套覆盖授权、活动、商品、转链、达人跟进、操作日志和后续数据复盘的 SaaS 管理系统。

当前阶段的目标不是“做一个商品查询页面”，而是完成最小可运行的商品主链路闭环。

## 当前阶段

当前项目处于：

> Mock-first 阶段，第三方接口按官方文档参数模拟，内部业务流程真实落库。

这意味着：

- 第三方抖店 / 百应能力先通过 Mock 提供
- 系统内部状态、日志、页面链路必须真实可验证
- 后续切换真实 SDK 时，只替换 Gateway 实现层

## 当前主目标

当前主目标为：

`授权 -> 活动 -> 活动商品入库 -> 商品审核 -> 绑定活动 -> 转链 -> 达人跟进 -> 操作记录 -> 页面统一展示`

## 核心工程原则

1. 前端只调用系统内部 API
2. Controller 不区分 Mock / Real
3. Service 不区分 Mock / Real
4. 第三方能力统一通过 Gateway 抽象
5. Mock Gateway 和 Real Gateway 必须暴露一致能力
6. Mock 数据可以是假的，但业务状态和日志必须真实落库
7. 当文档与代码冲突时，以当前代码可验证事实为准，并回写文档
