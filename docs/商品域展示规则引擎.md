# 商品域展示规则引擎

## 规则版本

当前引擎版本 `DISPLAY_RULE_VERSION = 3`，落库表 `product_display_rule_version`。

## 核心规则

1. 同一 `product_id` 仅一条 `DISPLAYING`（唯一索引约束）。
2. 保护期默认 3 个月，优先 `activity.months_of_protection`。
3. 保护期内普通候选不可覆盖；优势候选（更高佣金 / 更低服务费 / 投流 / 管理员强优势）可覆盖且不重置 `first_displayed_at`。
4. 保护期结束后按排序重选：强制展示 > 投流 > 佣金 > 服务费 > 上架时间 > relation_id。
5. 活动过期隐藏 `ACTIVITY_EXPIRED` 并重选；活动延期触发 `ActivityExtendedEvent` 后对该活动重算。

## 管理员干预

- `POST /api/admin/products/display/force-switch`
- `POST /api/admin/products/display/cancel-force`
- `GET /api/admin/products/display/audit-logs`

## 定时任务

`ProductDisplayRuleJob` 每小时执行 `reconcileAll()`，处理保护期到期、活动过期、强制展示到期。

## 事件

展示切换写入 Outbox：`ProductListedEvent`、`ProductHiddenEvent`、`ProductDisplayRuleAppliedEvent`、`ProductForceDisplayChangedEvent`。
