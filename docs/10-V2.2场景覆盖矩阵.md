# 10-V2.2 场景覆盖矩阵

更新时间：2026-04-21

## 一、核心业务链路

本项目当前重点覆盖以下五大核心业务场景：

1. **商品主链路**：从团长活动商品同步、系统内部初筛审核、招商负责人分配、到自动化生成推广链接的完整 SOP。
2. **订单自动化归因**：基于 `pick_source` 标识，将回流订单自动匹配至对应的商品、活动、达人及内部渠道负责人。
3. **寄样闭环管理**：支持从达人申请、内部审核、物流发货到签收回流的全生命周期管理。
4. **达人 CRM 保护**：实现达人公海/私海认领机制，通过保护期锁定防止内部撞单。
5. **数据驾驶舱**：提供归因准确率分析、招商业绩摘要及渠道推广效果看板。
