## 今日完成

### 1. 商品库主链路收口

商品库主链路 MVP 已稳定在如下状态流转：

`活动商品同步 -> PENDING_AUDIT -> APPROVED -> BOUND -> ASSIGNED -> LINKED -> FOLLOWING`

今天确认并收口的内容：

- `DouyinProductGateway` 已收口，业务层不再直接拼第三方商品 Mock 数据
- 商品统一主状态来源为 `product_operation_state.biz_status`
- 审核、绑定、分配、转链、达人跟进统一走状态机
- `product_operation_log` 已记录 `before_status` / `after_status`
- 商品详情已支持查看 `followRecords`
- 商品库阶段后续只做回归修复，不再扩主链路新功能

### 2. 达人跟进 MVP 并入主链路

已完成：

- `LINKED -> FOLLOWING` 首次跟进推进
- 跟进记录落 `talent_follow_record`
- 商品详情可回看跟进记录
- 跟进失败场景已补中文错误提示

### 3. 订单回流与归因 MVP 首轮跑通

本轮已完成后端闭环：

`promotion_link -> pick_source_mapping -> POST /orders/sync -> orders 落库 -> AttributionService 归因 -> GET /orders/unattributed -> GET /dashboard/summary`

已完成：

- `DouyinOrderGateway` 的 Mock 链路可返回可归因 / 无映射 / 无 `pick_source` 三类订单
- Mock 转链可生成 `pick_source_mapping`
- 订单同步允许未归因订单正常落库
- 新增未归因订单列表接口
- 新增订单摘要看板接口
- 订单列表排序已调整为最近更新时间优先

### 4. 前端联调与中文化修复

已完成：

- 订单工作台页面接入中文文案
- 历史订单页关键文案修正为中文

- 前端错误提示对 UUID / LocalDateTime 解析错误已做中文化
