## 当前执行原则

- 前端只调用内部 API，不直连第三方
- Service 层只依赖 Gateway 契约
- Test / Real 只在 Gateway 实现层切换
- Test 数据必须通过接口和数据库真实流转，不允许把演示结果写死在前端
- 当前演示基线优先保证稳定，不随意破坏核心闭环
