## 当前执行口径

- 前端只调用内部 API，不直连第三方。
- Service 层只依赖 Gateway 契约，不感知 Mock / Real 实现切换。
- Mock 数据必须真实走接口和数据库，不允许把演示数据写死在前端。
- 订单归因、寄样自动完成、MockConsole、Mock 物流状态推进当前已冻结为演示基线，后续只修 bug，不做大改。
