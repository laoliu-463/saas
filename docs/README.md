## 当前最重要的执行原则

- 前端只调用内部 API，不直连第三方
- Service 层只依赖 Gateway 契约
- 本地 Mock 与 Real 只允许在 Gateway 实现与配置层切换
- 本地联调必须先看业务，再看页面，再看接口，再查数据库，再看日志
- 未经明确要求，不接真实抖店 / 抖音 / SDK

3. 清理 Mock 数据与本地噪音日志问题
