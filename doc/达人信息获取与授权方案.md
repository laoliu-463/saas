# 达人信息获取与授权方案（合规主路径）

## 1. 结论
- 达人 CRM 可继续建设。
- 生产环境主路径应为：`官方授权 API + 达人主动提交 + 内部业务沉淀`。
- 公开页面采集仅保留为可配置辅助能力，默认关闭。

## 2. 数据来源优先级
1. `OFFICIAL_API`：抖音开放平台授权接口（首选）。
2. `MANUAL`：达人/渠道人工录入（兜底）。
3. `INTERNAL_BUSINESS`：寄样、订单、服务费等系统内沉淀。
4. `THIRD_PARTY`：合规第三方数据源。
5. `PUBLIC_PAGE`：公开页面低频辅助（默认关闭，不作为主链路）。

## 3. 系统策略
- 新增达人：不依赖公开页面采集作为成功前置条件。
- 刷新达人：当 `TALENT_PUBLIC_PAGE_CRAWL_ENABLED=false` 时拒绝公开页面采集刷新请求。
- 前端提示：引导优先走“达人授权 / 人工补录”路径。

## 4. 配置项
- `talent.data.public-page-crawl-enabled`（默认 `false`）
- 环境变量：`TALENT_PUBLIC_PAGE_CRAWL_ENABLED`

## 5. 里程碑建议
- V0.5：人工录入 + CRM 主流程闭环。
- V1.0：接入官方授权 API（OAuth + scope 管控）+ 人工兜底。
- V2.0：授权中心、字段来源审计、独家达人自动化增强。
