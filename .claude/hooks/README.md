# Hooks 守卫

[V1 必做] Hooks 是提交前、测试前、real-pre 前的检查清单。当前为文档化守卫，后续可接入 Claude / Codex / CI。

| 守卫 | 触发点 | 阻断条件 |
| --- | --- | --- |
| 变更前守卫 | 开始修改前 | 未读范围、领域、流程、验收 |
| real-pre环境守卫 | 任何 real-pre 操作前 | mock 环境、test flag 开启、无证据目录 |
| 测试前检查 | 跑测试前 | 端口/环境/依赖不清 |
| 文档同步检查 | 改实现后 | 对应 docs 未同步 |
| 提交质量门禁 | 提交前 | 无验证、无证据、冲突未登记 |
