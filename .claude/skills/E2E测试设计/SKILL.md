# E2E测试设计

description: 设计 test/mock 与 real-pre 的浏览器 E2E、API E2E 和证据路径。

## 步骤

[V1 必做] 读取 `README-e2e.md`、`docs/09-测试验收总览.md`、`docs/验收/E2E浏览器测试手册.md`。

[V1 必做] 明确脚本、角色、数据前置、API/SQL 补证、证据目录。

[V1 必做] test/mock 用 `npm run e2e:v1-p0`；real-pre 用 `npm run e2e:real-pre:p0`。

[V1 必做] real-pre 结论必须区分 `PASS`、`PASS_NEEDS_CLEANUP`、`BLOCKED`、`PENDING`、`FAIL`。
